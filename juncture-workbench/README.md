# Juncture Workbench

The Juncture Workbench includes 2 tools as an aid in creating visual essays.  The **editor** is a web-based text editor that provides a convenient means for previewing an essay during the authoring process.  The editor is also able to read and write Github files is a user has logged-in to Github from Juncture.

## Editor

`image Juncture_editor.png .right`

- The [Github file selector toolbar](zoomto/-28,-273,1158,1122/ac4dc01b) is used to select a Github file to be edited.  Pull-down menus are provided for selecting the name of the Github account owner, the repository hosting the Markdown file, the repository branch, and the path to the file relative to the repository root. 
- The [Markdown formatting toolbar](zoomto/-113,-116,834,808/58645b13) provides a convient way for adding common Markdown formatting tags to selected text.
- The [File actions toolbar](zoomto/241,-110,834,808/3a8152c9) provides controls for saving a file to Github, copying the Markdown text, opening the essay in a new tab, and toggling between the editor "Edit" and "Preview" modes,  Note that when opening the essay in a new tab the Github saved version is used rather than the browser version.
