# Juncture

Juncture is a free service for generating web pages from Markdown files hosted on GitHub.  Juncture extends the core Markdown language to add tags for easily adding interactive images, maps, and other visualizations to a page.  Juncture is often used to create visual essays that combine text narrative with interactive elements for adding depth and context.
